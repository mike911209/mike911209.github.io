【心得】2024 梅竹黑客松

Created2024-10-24|Review

今年的黑客松對我而言是充滿了驚喜的一屆。

從去年的做不完到今年的梅竹大獎第二名，我看到了我在這一年中的進步，也很感謝隊友的給力，讓我們在整個過程中都各司其職，效率非常高，在沒有偷卷的情況下，我們是真正意義上的黑客松，還能得到第二名，真的非常開心，現在回想起來還會覺得像是在作夢一樣。

黑客松 Day 0

今年的黑客松時程撞上專題準備期，加上到了大四其實會有點沒心思弄這些，幸好我們抽到了羅技組，原本還在賭爛他們到賽前一天才開放 api 讓我們不能偷卷，現在回想起來反而覺得很好，這樣就能專心準備其他事情。

賽前一天羅技舉辦工作坊，在新竹羅技辦公室，我走到交大搭接駁車，結果才發現我剛剛根本就有經過羅技辦公室 🙉。

羅技的工作坊很用心，他們請了很多工程師來看情況，也很耐心地等待沒有跟上的組別（其實是一直在等我，靠北），還幫我們準備了下午茶，重點是！還有有獎徵答！聽到的當下我就在想，羅技嘛，羅技會送甚麼，當然是滑鼠鍵盤阿，猛一看主持人手上拿著小盒子，好耶根本是滑鼠嘛，剛好我滑鼠又壞掉，當下馬上光速舉手回答問題，水啦，滑鼠++

這滑鼠超可愛而且很好用，拿到之後就一直用到今天，好開心。

黑客松 Day 1

黑客松當天我們到現場後，就照著工作坊完開會討論的開始實作，因為我們分工明確，幾乎都是 loosely coupled 的工作內容，所以我覺得我們進度很快，就一直在推進度。

羅技的工程師很常來關心我們，覺得他們超用心，在剛開始寫的時候有位大哥過來拍我肩膀：「怎麼樣？」我：「超穩👍」，但當下的我根本才剛開始寫，笑死。

特別要提一下的是第一天中午的午餐真的完全沒味道，吃的好痛苦。

默默的就到了晚上，今年有提供過夜場地，看下午的進度原先以為不用過夜，結果因為 multi-agent 遇到了點問題，最後還是只好繼續留著寫 code。

凌晨的時候真的是很慘，肚子超級餓，code 還一堆 bug，但梅竹黑客松很貼心，還有準備泡麵跟宵夜，吃一吃之後好像又有能量繼續寫了。

約莫在早上 6, 7 點左右，我終於把前後端還有 multi-agent 串起來，看到 work 的當下真的很開心，也覺得哇，感覺有機會到羅技組前三名哦。

黑客松 Day 2

結果我因為完全沒睡覺，早上大腦被強制關機了一小時左右，醒來後隊友已經在準備 present 了，真的超扛。

結果 present 前出大事，我們的 code 突然出 bug，前端會一直 render error，當下真的超緊張，覺得完蛋，如果修不好，又要上台丟臉了。

不知道是甚麼原因造成早上可以現在不行，所以我直接 git revert，再修改部分的 code，幸好有在時限前修好，也讓我們 demo 時候可以 live demo。

（附上我們作品的圖片

Demo 時要全英文 present，幸好我們有位隊友超 carry，英文很順，present 跟回答評審都回答得超好，在 demo 結束的當下，評審們看起來都超滿意，甚至在我們回去後還請我們再 demo 一次給剛剛沒有看到的羅技員工。

整個黑客松的過程中，心情最激動的時刻還是在 Line bot 公告中看到我們隊伍的名字

當下真的整隊歡呼，雖然知道我們表現的很好，但能受到這樣的肯定還是覺得非常不可思議，甚至我們在 present 前還完全 run 不起來，現在回想起來還是覺得有點如夢似幻。

上台後我們就盡我們所能，幸好 demo 過程中也沒有出問題（錄音功能在前面 2 次 demo 都出問題，結果是麥沒連上 🙉），順順利利的就下台了。

下台後羅技團隊跑來找我們聊天，真的覺得羅技很用心，而且熱情，我們還一起拍了一張大合照。

跟羅技團隊聊了很多天，默默的就到了最後公布梅竹大獎的時間。

第二名公布時，我們其實覺得有點可惜，所以當下沒有像其他組別一樣歡呼，但羅技團隊幫我們歡呼了，他們特別為我們留下來，在後面幫我們歡呼加油，真的很感動。

最後還收到羅技送我們的耳機，甚至他們還邀請我們 2 週後到羅技辦公室再 present 給他們的 leader 一次，很高興能這樣被重視。

在這次的黑客松中有許多出乎意料的部分，像是有人的電腦不能跑羅技的硬體、demo 前 code 壞掉、當然還有得獎，發生了好多好多難忘跟意想不到的事情。很感謝我的隊友們，都非常給力且盡責，大家幾乎都睡 1, 2 小時而已，我們的合作真的很順利。

在 2 天的期間內，收穫非常多，還有被詢問實習的機會，很慶幸今年有參加黑客松，讓我的大四不是只有準備專題跟讀書而已，雖然這 2 天非常累，但真的造就了非常美好的回憶。
