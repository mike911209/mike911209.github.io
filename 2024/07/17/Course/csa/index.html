【修課心得】計算機系統管理

Created2024-07-17|Course

課程相關

課名：計算機系統管理 Computer System Administration
科號：CS 342700
老師：張君天
課本：教授自編講義
課別：資工系選修（非ABCD類）
學分：3
涼度：★★★★★（滿分五分，空心代表半分）
甜度：★★★★★（滿分五分，空心代表半分）

課程內容/簡介

管理計算機系統，作業系統使用freeBSD，對於coomand line interface不熟的修這堂課會對Unix系統有初步的認識，非常推薦資工系大三來修。
課程前期教學網路架設，牽涉到一些計網概的知識，中期會教Apache server，讓學生自己架DNS等等，課程後期會教一些硬碟的管理。

教學進度如下：

  1. virtual machine
  2. IP setting
  3. User administration
  4. Firewall
  5. SSH
  6. Apache
  7. cgi
  8. DNS
  9. process management
  10. SSL

以上是課程前中期的內容，後期的因為老師的課程網頁目前已關閉，我沒有記錄到，就不列出。

上課方式

第一堂上課時老師問大家想學什麼，許多人舉手，老師評估之後如果與這堂課相關就會納入課綱。
16周課程，實體加線上，會錄影但不公開，僅做教學紀錄。錯過一堂課將可能會導致後面相關的課程直接跟不上，建議整學期出席並跟上上課進度。
上課教材與課綱都在老師架設的課程網頁內，上課教材比較像是老師的筆記，不會有很詳細的操作指示。
與一般課程不同，上課期間完全是實作，老師會帶大家設定虛擬機、安裝package、調整設定等等，非常基礎的計算機系統管理。老師的期望是大家之後進到公司面對一台真正的機器時可以知道怎麼設定。
老師上課時會操作虛擬機投影到大家的螢幕上，學生要自己跟上老師的教學內容，老師打指令非常快，一旦錯過可能就會直接跟不上，要適時舉手跟老師詢問，或者如果旁邊有個超強的linux超算大神那就沒問題。
老師很耐心，在課堂中間跟不上只要舉手跟老師說老師都會願意重講，但如果沒人舉手老師default的設定就是大家都有跟上，但實際底下可能哀鴻遍野。
在課堂中間時常會遇到各式各樣的問題，老師會直接現場解決，非常厲害，可以觀摩大神怎麼觀察、搜尋、解決問題的。

作業

有6還7項作業，都是當天上課期間可以直接完成，也不難，如果不會的話直接向老師求助老師也會一步步帶學生完成。作業期限通常給2周非常夠，沒有甚麼特別狀況都可以拿100分。

作業憑我印象大致如下：

  1. 設定ssh連線
  2. 架設apache web server
  3. 取得憑證
  4. 架設mail server
  5. 好像cgi還是DNS
  6. 忘了

考試 && 期末報告

期中、期末各佔40%

期中分術科與學科，共90分鐘吧好像，考試期間可以上網搜尋，術科搜尋沒有幫助，學科可以靠google拿滿。
術科考上課計算機系統管理的內容，會給一點情境，跟學生說明現在虛擬機的狀況，讓我們將虛擬機的網路復原或者將架到一半的DNS server架好等等。術科考試內容我個人覺得很困難，建議花2天左右的時間研究術科考古，在考試期間會輕鬆非常多，但如果你是超強的linux超算大神那就完全沒問題，穩穩的100。
學科主要是考指令的功用跟一些非常基礎的東西，共33題，非常簡單。
總而言之，術科有練習考古的話期中總分應該可以拿到80左右

期末老師讓大家投票考試還是報告，最後是報告，老師偷偷跟我說考試的話他要架環境，其實比我們還累。
期末報告自己一組，選擇一個freeBSD中的package介紹。老師會在課程結束前約4周提醒大家，選好要報告的主題後跟老師說就可以，但如果主題太簡單老師會指定要做的內容。
報告當天實體或者google meet出席皆可，準備簡報以及虛擬機，在報告期間展示以及介紹package，老師分數都給很高，我一整排看下來都有個8、90。

結論

對command line interface不熟的非常推薦來修這堂課，修完後會有很好的Unix基礎。也很推薦資工大三來修，這個時期通常是對計算機架構了解但又沒有實際管理操作過，修完後至少能會一些command line跟環境架設。我自己在修完後有感覺到對於問題的處理以及環境架設等等有變更熟悉，不會像以前一樣感到很頭痛。
課程期間要跟上老師的節奏偏累，但課程外都不用花任何時間，是非常划算又能學到東西的一門課，甜涼好課推推。

給分

作業 20% +
期中 40% +
期末 40% +
= 100%
老師最後有來個大調分，很多在接近A+邊緣的同學都被老師送上去，佛心公司。

老師的喜好、個性

外號張天神，什麼都會，什麼都懂，似乎是資工系計中的老大。人非常好，在虛擬機中遇到任何問題向老師詢問，老師可能會直接到座位旁邊幫忙看，現場將問題車過去，像是1+1一樣簡單。

加簽

給，似乎是全簽，在第一、二堂課過後很多人沒跟上就會退掉，殊不知這堂課又涼又甜，所以如果沒簽上也沒關係可以等加退選再加。

成績

總成績/班上排名：94(A+) 1/24
T分數：55.65（看T分數就能知道有多甜）
成績分布：
