【修課心得】軟體設計與實驗

Created2024-07-17|Course

課程相關

課名：軟體設計與實驗
科號：CS 241001
老師：吳尚鴻
課本：教授自編講義
課別：資工系系定必修
學分：3
涼度：★（滿分五分，空心代表半分）
甜度：★★★（滿分五分，空心代表半分）

建議先修課程

計算機程式設計

課程內容/簡介

教學軟體設計以及UI/UX design時的一些想法，還有做interview跟usability test等等。技術部分則是先教學基本的HTML/CSS，教Javascript，後面開始React跟backend，最後面會帶到一點intelligence。

教學進度如下：

  1. 軟體設計：

    • User needs
    • feature design
    • fast prototyping, and usability test

  2. 技術課程：

    • HTML/CSS
    • javascript
    • reactive programming
    • backend development
    • web/app intelligence

上課方式

有自己的教學網頁。
上課不點名，內容分為2大topic，軟體設計以及技術課程，禮拜二上課時各花一半的時間講解，但老師的上課時間通常是無視課表直接上滿2.5小時，經常被管理員關心哈哈哈。
技術課通常是提取重點出來講，剩下的回去看每週4、5個小時的教學影片。
老師課堂很喜歡大家討論跟提出問題，老師也會很耐心的跟台下互動，上課方式很輕鬆幽默，偶爾回答老師的問題可以加分。
設計課程很有趣，老師懂很多經驗也很豐富，可以聽到很多故事，但有趣僅限聽聽而已，當設計部分跟成績扯上關係然後要求還不清不楚的時候這部分就很難受了，在後面的部分我會敘述原因。

Youtube教學影片教學很清晰易懂，印象中沒有字幕，但聲音很清楚。
影片通常講一次就聽懂，但內容真的超級多，而且很常會有assign reading，每週都要花將近10～15小時的時間固定在看影片跟練習。
教學影片內容很舊，gitlab上的資源經常跟影片內容對不上，助教資源公不公布也不會講，很常都是Lab前2天才給，一整個難受。

禮拜四是Lab，使用gitlab，有需要當場寫出來的也有一個禮拜的小組或個人作業，當場如果沒寫出來當天00.前交分數打6折。

考試 && 作業

Lab共有12次，一半的都很簡單。
去年似乎是做android，跟今年網頁設計不同。
Lab的考試內容有高機率跟教學影片的作業一樣，推薦可以把影片的作業寫一寫，但有時候寫完就會發現助教後來公布template，這種情況滿常發生，resources都很晚公佈。
小組作業部分我們這組是一個人寫完一次Lab，如果要拆開分工的話或許會出現互相拖到的問題，redux那次好像不少組分工後卻沒寫完，可以斟酌分工。
資料庫的Lab template有bug（感謝 B9 補充，不是template的問題，是資料庫設定時出錯），導致很多人被AWS發了很可怕的帳單，如果之後還是用AWS的話要每天小心帳單。
後期的Lab非常簡單，有時候等gitlab loading的時間都比寫code還久，只有前期的color game算很難，之後的分數都滿好拿，Lab的分數很甜。

沒有期中期末，但有三次產品的demo，要做的內容就是設計課老師講的那些，bottom up的設計出一個產品，老師希望是App而不是遊戲，要滿足使用者的需求。
前2次demo都不是打code，內容可能是interview 或usability test等等，最後一次才是實作，基本上我的figma比Javascript 還熟。
demo分數很玄，真的很玄，我們這組感受到的是甜甜分數，但其他組也有40、60分的，簡報評分規則很簡陋，基本上就是通靈，所以我們這組是老師上課有提過的所有東西我們都全做，分數就大概90。
很多東西規則不清楚，問助教後助教會再新增規則，助教自己也不是很清楚這部分的評分跟執行方式，討論區基本上要當聖經在讀，就一個必修課程而言，身為學生我的感受滿差的。

第三次demo教授會給各組4分鐘的時間demo產品，之後會有6分鐘的QA時間，教授或助教會依據第二次demo時報告的flow去詢問你的App等等。
小組四個人以上產品要有後端。
最後會有人氣王票選前三名有很多額外的加分。

demo會有匿名小組互評，會影響成績很多，好像有組別發生心結大會。

總體而言，學會很多開發產品的細節，也寫了很多網頁，可惜評分規則不清楚以及resources的陳舊，但修完之後確實trace code的能力變很強，寫網頁時也很開心，挺喜歡這堂課的，但還是有許多問題，希望之後會有所改善。

給分

1、2次demo 各10% +
第三次demo 20% +
Lab 60%
= 100%
上課回答問題可以額外加總分
demo分數可以破百，會有額外bonus

老師的喜好、個性

型男帥氣老師，上課幽默風趣，但也很瘋。還記得清華炸彈危機那時老師說想走的可以走，我捨命你隨意。
很樂於回答問題，也很喜歡大家上課提出問題跟老師互動，是很好的老師。
經驗跟知識都很豐富，了解的領域橫跨多個維度，有時候會讚嘆老師怎麼懂這麼多。

加簽

給，詳細規則我不清楚。
